# krew
kubectl plugin manager

## kubectl ctx
```bash
# install using krew
kubectl krew install ctx
# using ctx, check cluster
kubectl ctx

kubectl ctx docker-desktop
## Switched to context "docker-desktop".
```
## kubectl ns
```bash
# install using krew
kubectl krew install ns

kubectl ns
## default
## istio-system
## kube-node-lease
## kube-public
## kube-system
## local-path-storage

kubectl ns local-path-storage
## Context "docker-desktop" modified.
## Active namespace is "local-path-storage".
```
